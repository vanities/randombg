# wsg-random-bg
gets a random wallpaper from /wg/


## mac installation

### install on mac:
```bash
sudo make mac
```

### uninstall on mac:

```bash
sudo make clean-mac
```
### check:
```bash
launchctl list
```

### stop:
```bash
launchctl stop bgmac
```

## linux installation


### install on linux:

```bash

sudo make linux

```

### uninstall on mac:

```bash

sudo make clean-linux

```
