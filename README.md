# randombg
gets a random wallpaper from /wg/ on a timer for mac and linux


## mac installation

### install on mac:
```bash
make mac
```

### uninstall on mac:

```bash
make mac_uninstall
```
### check:
```bash
make mac_check
```


## linux installation


### install on linux:

```bash

sudo make linux

```

### uninstall on linux:
```bash

sudo make linux_unintstall

```
